About the Project

The purpose

To predict U.S. rates of respiratory infection amongst adults on a county-level by analyzing several air quality metrics

The Plan

  1. Identify data sets
  2. Clean data sets, removing unneeded data
  3. Develop ML models
  4. Record stats from target cities
  5. Compare model outputs to city targets
  6. Create charts in Tableau
  7. Embed visualizations in a website
